# Definition Of Done 

The user will be able to interact with the SWAN Wave modelling program, using a graphical user interface (GUI). This interface will allow the user to select all the parameters required and an output file will be generated based on these parameters. The user will then be able to select all the required data files and be notified if the input ot data files are missing. 
The user will then be able to run the SWAN program by clicking a button, which will automated the call to the SAWN program. If any errors occur the GUI will represent them in its own console window; a log file will also be created.
After the SWAN program has completed the modelling, the GUI will notify the user and present a raw text representation of the data.
